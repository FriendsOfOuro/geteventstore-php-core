Roadmap
-------

Development started in April 2014. Not ready for production use.

API can currently:

- Read from streams
- Navigate streams
- Read events
- Write to streams
- Delete streams

Symfony 2 integration
---------------------
[EventStore Client Bundle](https://github.com/dbellettini/eventstore-client-bundle) integrates this project in Symfony 2
